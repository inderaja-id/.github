# Inderaja Indonesia

**Inderaja** (from Bahasa Indonesia, _Penginderaan Jauh_), means _Remote Sensing_.



## About Inderaja Indonesia

**Note:** This is an **independent, community-driven educational initiative** created and maintained by [dmxsan](https://github.com/dmxsan). It is not affiliated with any government agency, university, or official remote sensing organization in Indonesia.

Our mission is to share open-source learning resources and tools with the Indonesian RS community as a grassroots contribution to knowledge democratization.
